## 📐 Опис
Цей проєкт створений для побудови макету згідно 

## 🚀 Технологічний стек
- **React** – сучасна бібліотека для побудови інтерфейсів
- **Vite** – швидкий інструмент для розробки та збірки
- **Tailwind CSS** – утилітарний CSS-фреймворк для гнучкої стилізації
- **TypeScript** – типізація для більшої надійності та зручності
- **Lucide Icons** – набір сучасних SVG-іконок
